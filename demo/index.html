Key Decisions & Tradeoffs

• Python CLI vs web app: CLI chosen for simplicity and CI/CD compatibility. Web app would require backend + auth.
• Self-contained HTML: No hosting required, easy to share via email or Slack.
• Chart.js via CDN: Keeps the codebase simple. Trade-off: requires internet for charts to load.
• Health score formula: Weighted toward churn (most impactful metric). Benchmarks from B2C subscription industry data.
• Bar charts for Revenue/Customers: Daily data aggregated monthly still looks better as bars than spiky lines.
