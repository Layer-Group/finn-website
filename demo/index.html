Wat RC Pulse doet

📊

Health Score

0–100 score met letter grade (A–F) op basis van churn, MRR groei en refunds.

💰

KPI Dashboard

MRR, ARR, actieve subscriptions en revenue — realtime, in één oogopslag.

📈

Trend Charts

4 interactieve grafieken: MRR trend, subscription groei, revenue en momentum.

🎯

Benchmarks

Jouw churn en groei vergeleken met industrie-gemiddelden voor B2C apps.

🔒

Geen server

Volledig client-side. Data gaat direct naar RevenueCat API — niets wordt opgeslagen.

⚡

CLI Tool

Genereer HTML rapporten vanuit je terminal. Integreer in CI/CD workflows.

Health Score systeem

RC Pulse beoordeelt je subscription business op churn rate, MRR groei en refund ratio.

A 90–100 Excellent
B 80–89 Good
C 70–79 Fair
D 60–69 Attention
F <60 Critical

CLI Tool

Dezelfde health score en charts, maar als HTML rapport vanuit je terminal.

# Install
pip install rc-pulse

# Genereer rapport (opent in browser)
rc-pulse report --api-key sk_jouw_key

# Sla op voor CI/CD
rc-pulse report --api-key $RC_API_KEY --output report.html --no-open
